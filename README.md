
# 📝 CV Projet — README

## 🎯 Présentation du projet

Ce projet consiste en la création d’un **CV en ligne**, développé uniquement avec **HTML**, **CSS** et au minimum une fonctionnalité **JavaScript**, puis hébergé via **GitHub Pages**.

Pour le style, j’ai choisi d’expérimenter une nouvelle approche en utilisant le **CSS de W3Schools**, afin de bénéficier de composants simples, légers et faciles à mettre en forme.

👉 **Lien vers le site en ligne (GitHub Pages)** :  
https://brunomartinsconstantino.github.io/CV_Projet/

👉 **Lien vers le dépôt GitHub** :  
https://github.com/BrunoMartinsConstantino/CV_Projet

## 🧪 Technologies utilisées
- **HTML5**  
- **CSS3** (avec éléments de *W3.CSS* de W3Schools)

## 🔀 Workflow Git utilisé
Pour organiser le développement du projet, j’ai mis en place un workflow Git simple.

### 🌱 Branches principales
- **main** – version stable, déployée via GitHub Pages.
- **dev** – zone de développement principale.

### ⭐ Workflow des features
1. Création d’une branche feature depuis **dev** :
   ```
   feature/nom-de-la-feature
   ```
2. Développement de la fonctionnalité.
3. Merge de la branche feature dans **dev** une fois terminée.
4. Quand **dev** est stable, création d’une **Pull Request vers main**.
5. Le merge dans **main** déclenche la mise en ligne automatique du site.

---

## 🚀 Déploiement
Le site est déployé automatiquement via **GitHub Pages**, configuré pour pointer sur la branche `main`. Chaque mise à jour de `main` met en ligne instantanément la nouvelle version du CV.
